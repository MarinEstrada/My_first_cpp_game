#  My first cpp game

My first c++ game! I created a Pong game by following a tutorial by Dan Zaidan where I learned about the basics of game development including modularizing game mechanics and platform specific code as well as mechanics like the physics involved simulating movement. I built the game for use in Windows OS and used Claude to translate my code into an emscripten version to run the game locally online in github pages as well as to integrate the Pixelify Sans font into the game.

Overall it was a fun little project! Looking forward to using what I learned to create my own games for fun :) You can play it here: [https://marinestrada.github.io/My_first_cpp_game/](https://marinestrada.github.io/My_first_cpp_game/)

May return to this later to integrate accesibility settings such as input sensitivity and the ball size.
